🔊 Sagalee qindeessuu

Sagalee sirriitti dhageeffachuun barachuu keessatti barbaachisaa dha. Appiin kun sagalee browser yookiin bilbila kee (Web Speech API) fayyadama. Sagaleen Faransaayii meeshaalee hedduu irratti argama — yoo hin jiraanne, gara gadii hordofi.

🤖 Android
  1. Banaa Qindaa'inaa → Gargaarsa → Sagalee uumamaa (yookiin « sagalee » barbaadi).
  2. Injinii filatamaatti, Google Text-to-Speech filadhu, Play Store irraa haaromsi.
  3. ⚙️ → Daataa sagalee buusi → Faransaayii (Fraansi) buufadhu.
  4. Appii keessatti deebi'i, fuula haaromsi.
💡 Samsung irratti : Qindaa'inaa → Gargaarsa → TTS. Sagleeleen dabalataa sanuma irraa buufamu.
🍎 iPhone / iPad (iOS)
  1. Banaa Qindaa'inaa → Gargaarsa → Dubbii → Sagalee.
  2. Afaan filadhu, sagalee filadhu, ⬇️ cuqaasi — Fooyya'aa buufadhu.
  3. Appii Safari keessatti bani (iOS irratti filatamaa) — 🔊 jalqabaaf cuqaasi, iOS hayyama sagalee gaafata.
  4. Sagaleen yoo hin bane — qaaccee callisaa (cinaa irratti) mirkaneeffadhu.
💡 Onglet 🎤 Irra deebi'i Safari iOS 14.5+ fi Chrome Android irratti qofa hojjeta — Firefox mobile irratti hin hojjetu.
💻 Kompiyuutara (Windows / Mac)
  1. Chrome yookiin Edge filatamoo dha — sagalee gaarii fi dubbii beeksisuu ni deeggaruu.
  2. Windows : Qindaa'inaa → Sa'aa fi afaan → Dubbii → Sagalee dabalata → Faransaayii (Fraansi) buufadhu.
  3. Mac : Filannoo Sirna → Gargaarsa → Dubbii → Sagalee sirna → Thomas (FR) buufadhu.
  4. Buufannaa booda browser haaromsi — sagleeleen haaraan fe'amti.

🗺️ Comment ça marche ▼

Cette appli vous guide pas à pas pour apprendre l'Oromo, à votre rythme, sans inscription ni abonnement.

Elle est organisée en deux niveaux :

  • 📚 Niveau 1 — Vocabulaire (32 thèmes) : Cartes Flash, Lexique, Quiz, Répétition orale
  • 💬 Niveau 2 — Dialogues (16 mises en situation) : scènes de la vie réelle à écouter, comprendre et répéter
🚀 Lancement → 📚 Modules → 🃏 Cartes → 🎯 Quiz → ⭐ Étoiles

Le Quiz et les Étoiles ⭐ : après les cartes, un Quiz (3 à 10 questions selon la taille du module). Choisissez la bonne réponse parmi 4 options.

  • ⭐ : ≥ 50% → module validé ! · ⭐⭐ : ≥ 75% · ⭐⭐⭐ : 100% 🎉
  • Les étoiles ne diminuent jamais — si vous rejouez et faites mieux, le compteur monte ; si vous faites moins bien, il ne bouge pas.

Ce Guide est accessible à tout moment via le bouton ❓ dans la barre de navigation en bas.

💡 Commencez par le Niveau 1 — les dialogues du Niveau 2 seront plus faciles ensuite !

🔊 L'audio ▼

🔊 Configurer son audio

Bien entendre les mots, c'est essentiel ! L'appli utilise la synthèse vocale intégrée à votre téléphone ou navigateur (Web Speech API). Si le son est absent, robotique ou dans la mauvaise langue, suivez le guide ci-dessous.

🤖 Android
  1. Ouvrez Paramètres → Accessibilité → Synthèse vocale (ou cherchez « synthèse » dans la barre de recherche).
  2. Dans Moteur préféré, choisissez Google Text-to-Speech et mettez-le à jour dans le Play Store.
  3. Appuyez sur ⚙️ → Installer les données vocales → téléchargez la voix Français (France).
  4. Revenez dans l'appli et actualisez la page.
💡 Sur Samsung : Paramètres → Accessibilité → TTS. Les voix supplémentaires se téléchargent depuis ce même menu.
🍎 iPhone / iPad (iOS)
  1. Ouvrez Réglages → Accessibilité → Contenu énoncé → Voix.
  2. Sélectionnez la langue cible, choisissez une voix et appuyez sur ⬇️ pour la télécharger en qualité Améliorée.
  3. Ouvrez l'appli dans Safari (recommandé sur iOS) et tapez une première fois sur 🔊 — iOS demande une interaction avant d'autoriser l'audio.
  4. Si rien ne sort, vérifiez que le bouton silencieux (interrupteur sur le côté) est bien désactivé.
💡 L'onglet 🎤 Répète n'est disponible que sur Safari iOS 14.5+ et Chrome Android — pas sur Firefox mobile.
💻 Ordinateur (Windows / Mac)
  1. Chrome ou Edge sont recommandés — ils embarquent de bonnes voix et supportent la reconnaissance vocale.
  2. Windows : Paramètres → Heure et langue → Parole → Ajouter des voix → installez Français (France).
  3. Mac : Préférences Système → Accessibilité → Contenu parlé → Voix système → téléchargez Thomas (FR).
  4. Redémarrez le navigateur après l'installation — les nouvelles voix sont détectées au chargement.

Si l'audio ne fonctionne toujours pas, le problème vient probablement d'un réglage spécifique à votre modèle. Les équipes support peuvent vous aider :

🎧 L'audio en Oromo : la Cascade de Voix

L'Oromo (Afaan Oromoo) s'écrit avec l'alphabet latin (le Qubee). Comme la voix native Oromo n'est pas encore préinstallée sur tous les appareils, l'application utilise un système intelligent de cascade phonétique — si le niveau 1 est absent, elle descend automatiquement au suivant.

🟢 Priorité 1 🇪🇹 Oromo [om-ET] L'idéal absolu — prononciation 100% authentique
🔵 Priorité 2 🇸🇴 Somali [so-SO] Famille couchitique — maîtrise les voyelles longues (haaraa, osoo…)
❌ Haaraa → [ara]✅ Haaraa → [haaa-raaa] ❌ Osoo → [ozo]✅ Osoo → [osooo]
🟡 Priorité 3 🇪🇹 Amharique [am-ET] Éthiopie — musicalité d'Afrique de l'Est, consonnes éjectives
❌ Caffee → [café]✅ Caffee → [tcha--ffé] ❌ Xalayaa → [ksalaya]✅ Xalayaa → [t'alavaa]
🟠 Priorité 4 🇳🇬🇰🇪 Haoussa / Swahili [ha-NG / sw-KE] Voyelles pures — le E fait [é], le U fait [ou], sans avaler les finales
❌ Bari → [barin]✅ Bari → [ba-ri] ❌ Mucaad → [mycad]✅ Mucaad → [mou-caad]
🔴 Priorité 5 🇪🇸🇮🇹 Espagnol / Italien [es-ES / it-IT] Bouclier anti-français — U=[ou], CH=[tch], zéro voyelle nasale
❌ Mana → [mâna]✅ Mana → [ma-na] ❌ Gutu → [gy-ty]✅ Gutu → [gou-tou]
💡 Au premier clic 🔊, un bandeau discret vous indique quelle voix est active — ex. « 🎤 Audio Oromo configuré avec la voix : Somali ». Vous savez ainsi avec quel accent travaille votre oreille !

🔄 Réinitialiser un module ▼

Envie de recommencer un module à zéro ? Vous pouvez effacer vos étoiles et repartir de la base.

  • Depuis l'écran Modules, repérez le thème déjà complété — le bouton 🔄 Recommencer apparaît directement sur sa carte.
  • Appuyez sur ce bouton — une confirmation vous sera demandée.
  • Vos étoiles sont supprimées — rejouez le Quiz pour en regagner.
